# Darvin Code — Product Context

## Brand
**Name:** Darvin Code  
**Type:** Software solutions studio  
**Focus:** Hospitality industry (hotels, resorts, villas, hospitality brands)  
**Register:** Brand (marketing/landing page — design IS the product)

## Users
Hospitality decision-makers: GMs, Owners, Digital/Tech Directors at boutique hotels, luxury resorts, villa operators, hotel groups and collections, hospitality management companies.

## Product Purpose
Communicate premium positioning, studio credibility, and hospitality-native expertise. Convert visitors into qualified project enquiries.

## Brand Voice
Precise. Warm. Credible. Not corporate. Not agency-hypey. Speaks like a senior consultant who has lived inside hospitality operations.

## Anti-references
- Generic SaaS startup landing pages
- Dribbble over-designed aesthetics
- Excessive gradients / glassmorphism
- Template-based layouts
- Overly corporate or cluttered design

## Tone
Editorial. Confident. Direct. Hospitality-informed. Craft-obsessed.

## Typography
- **Display:** Urbanist (geometric sans, variable weights 300–900) — primary voice
- **Accent serif:** Spectral (italic cuts for emotional moments, pull quotes, contrasting headlines)
- Scale: modular with fluid `clamp()`, strong weight contrast

## Color
- Background: `oklch(97% 0.008 65)` — warm off-white
- Surface: `oklch(94.5% 0.012 55)` — warm beige
- Dark: `oklch(19% 0.010 55)` — charcoal (used for dark sections)
- Accent: `oklch(71% 0.105 42)` — soft coral / peach
- Text: graduated oklch scale from ink to muted

## Tech Stack
- Next.js 14+ (App Router)
- TailwindCSS v4
- Framer Motion
- GSAP (available)
- Lenis smooth scroll (@studio-freight/lenis)
- TypeScript

## Sections
1. Hero — full viewport, canvas particle animation, strong typography
2. Services — editorial accordion list
3. Featured Work — asymmetric project cards with CSS/SVG art direction
4. Why Choose Us — dark section with pull quote + differentiators
5. Process — 4-phase table layout
6. Contact CTA — form + info columns
7. Footer — minimal
